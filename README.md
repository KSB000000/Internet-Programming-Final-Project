
# Start

프로젝트를 실행시키기 위해서 VSCODE 를 열고 터미널을 2개를 준비합니다.

첫 번째 터미널에서는 
```
node server.js
```
두 번째 터미널에서 
```
mvn clean install
```
을 실행시킨 다음에
```
mvn spring-boot:run
```
을 실행시킵니다.

모두 실행 시킨 다음
